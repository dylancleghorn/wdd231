Project

Individual Website Project · WDD231

Site Name

Proposed Name: Scripture Speed Reader

This name clearly communicates what the site does: it helps users read scriptures at a controlled, configurable speed. The phrase "Speed Reader" mirrors the primary interaction in the app, where text is displayed word-by-word in the reader stage at a chosen WPM rate.

Purpose

The Scripture Speed Reader site provides a focused, distraction-reduced environment for scripture study using a timed, word-based reading experience. The site will:

  • Offer a scripture library where users can select a book and chapter.
  • Display scriptures one word (or phrase) at a time in the central reader stage.
  • Allow users to set and adjust reading speed in words per minute (WPM).
  • Track chapter completion and visually mark finished chapters.
  • Encourage consistent daily scripture reading with simple tips and goals.

Audience Scenarios

  • Scenario 1: "How can I pick a chapter from the library and read it word-by-word at a specific WPM so I don't get distracted or lose my place?"
  • Scenario 2: "Can I see which chapters I've already finished so I can track my progress through a book of scripture?"
  • Scenario 3: "Is there a way to pause, restart, or slow down the reader if the text is moving too fast for me?"
  • Scenario 4: "Can I change font size or theme settings so the scripture text is comfortable to read on my phone at night?"

Color Scheme

The Scripture Speed Reader app already uses a dark, high-contrast palette designed for extended reading in low-light environments. The colors below are the core tokens used across the app and this site plan.

Background · --bg
Hex: #0e0f12
Used as the overall page background behind the library, reader stage, and this site plan. It keeps the focus on the text and controls.
Foreground / Text · --fg
Hex: #f7f7fb
Used for primary text so scripture words and labels are bright and readable on the dark background.
Card / Panel · --card
Hex: #1b1e25
Used for cards, panels, and containers such as the reader stage, library sections, dialogs, and these plan sections.
Accent · --accent
Hex: #4ade80
Used for important highlights like active controls, key stats, or progress indicators (for example, completed chapter states).
Brand / Primary Button · --brand
Hex: #16a34a
Used for primary CTAs (such as the Start/Pause button) and interactive pills to visually reinforce actions the user should take.

Typography

The app uses a clean system UI stack so it feels native on most devices and keeps scripture text readable at various sizes.

  • Body Text & UI:
    system-ui, -apple-system, Segoe UI, Roboto, Arial, sans-serif
    Used for paragraphs, labels, buttons, chapter pills, and most interface text.
  • Headings:
    Same system-UI stack, but with larger sizes and heavier font-weight to differentiate sections such as "Library", "Reader", and settings headings.
